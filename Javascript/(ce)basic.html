    기본

    Modern mode & 주석

    javascript의 주석은 //로 표기된다. 여러줄 주석은 /**/

    use strict
    use strict는 현대적인 방식으로 스크립트가 작동하게 한다. 이것은 스크립트 가장 위에 선언해야 한다.
    하지않아도 상관 없지만 use strict를 선언했을 때 최신 javascript는 클래스와 모듈을 지원한다. 따라서 따론 선언해줄 필요 없다.

    변수

    대부분 javaScript는 imformation과 함께 작동한다.
    예를들어 온라인 상점에서 판매중인 상품이나 장바구니가 될 수 있고 채팅 application 같은 경우에는 사용자 또는 메시지등 이라고 할 수 있다.

    변수는 데이터에 대한 지정된 저장소이다. 변수를 만들려면 let을 사용한다. let을 사용하여 변수를 선언하면 관련된 메모리 영역에 저장된다. let을 사용한 변수 선언은 바꿀 수 있다. 값이 변경되면 이전 데이터가 변수에서 제거된다. javascript 같은 경우는 java와 같이 camelCase가 적용된다. 만약 불변의 변수를 선언하고 싶다면 const를 사용하면 된다. const를 사용하면 재지정할 수 없다. 이미 대중적으로 알려진 값이나 고정된 값은 모두 대문자로 표기하여 선언한다.

    데이터 유형

                        Javascript의 값은 8가지 유형이 있다.
                    숫자 
                    - 곱하기, 더하기, 나누기, 빼기 Infinity
                    - 큰 정수값은 정확도가 떨어진다. 고정된 6비트 저장소에 맞지 않으면 오류가 발생한다.
                    - 최근에 BigInt type은 임의 길이의 정수를 나탄기 위해 최근에 언어가 추가되었다.

                    string
                    - 문자열은 따옴표로 묶어야한다.
                    - 큰따옴표, 작은 따옴표, 백틱이 있다. " " ' ' 
                    - 백틱은 확장된 기능 인용 부호 이다. 변수와 표현식을 래핑하여 문자열에 embed할 수 있다.

                    boolean(논리 유형)
                    - 부울 유형에는 true,false가 있다.

                    Null 값
                    - null은 존재하지 않는 객체에 대한 참조 또는 널 포인터가 아니다. 
                    - null은 javascript에서 업음, 비어있음을 나타내는 뜻일 뿐이다.
                    - 즉 단일 값을 갖는 독립 실행형 유형이다.

                    undefined 값
                    - 값이 할당되지 않음을 나타낸다.
                    - 선언되었지만 할당되지 않은 경우
                    - 일반적으로는 권장하지 않는다.
                    
                    개체 및 기호
                    - object유형은 특별하다. 객체는 데이터 컬렉션과 더 복잡한 entity를 저장하는데 사용된다.

                    typeof 연산자와
                    - typeof 연산자는 피연산자의 유형을 반환한다. 
                    - 다른 유형의 값을 다르게 처리하거나 빠른 확인을 원할 대 유용하다.
                    - typeof는 함수가 아니라 연산자이다. 원래 typeof(X)이지만, 여기 괄호가 함수의 괄호가 아니다.

    interactive - 경고, prompt, 확인

                        1. alert
                        모달창을 통해서 방문자가 창을 처리할 때까지 페이지의 나머지 부분과 상호작용하거나 다음 버튼으 누르는 작업을 할 수 없음
                    
                    2.prompt   
                        두가지 인수를 허용한다.
                        텍스트 메시지 방문자를 위한 입력 필드 확인 취소 버튼이 있는 모달창을 보여준다.
                        prompt(title,[default])에서 title은 방문자에게 표시할 텍스트 default는 매개변수, 입력 필드 초기값
                        []부분은 선택사항이나 값이 없는 경우, undefined를 삽입해주는 것이 좋다.
                    
                    3.confirm
                        이 함수는 확인 및 취소라는 두개의 버튼이 있는 모달창을 보여준다.
                        결과는 true이거나 false인 것이다.
